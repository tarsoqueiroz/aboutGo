# aboutGo

My self study about Go language

## Install

> `https://go.dev/doc/install`

```Shell
wget https://go.dev/dl/go1.17.5.linux-amd64.tar.gz

sudo tar -C /usr/local -xzf go1.17.5.linux-amd64.tar.gz

ll /usr/local/
ll /usr/local/go

cd
pico .bashrc 
# Insert:
#         export PATH=$PATH:/usr/local/go/bin
source .bashrc 
go version
clear
```
